# Wireless Display Embedded System

This project demonstrates a wireless display system using LPC2148 microcontroller.

## Features
- Wireless communication
- LCD display output
- I2C protocol implementation
- EEPROM data storage

## Technologies Used
- Embedded C
- LPC2148
- I2C Communication
- LCD Interface

## Project Structure
- main.c : Main logic
- lcd.c : LCD driver
- i2c.c : I2C communication
- delay.c : Timing functions

How to Run

1. Open the project in Keil uVision IDE
2. Build (compile) the project
3. Flash the code into LPC2148 microcontroller
4. Connect LCD and required hardware components
5. Power ON the system
6. Observe output on LCD display

Working Principle

The LPC2148 microcontroller acts as the main controller of the system. It processes the incoming data and communicates with external devices.

The I2C protocol is used to interface with EEPROM for storing and retrieving data efficiently.

The processed data is then sent to a 16x2 LCD display, where it is shown to the user.

This system enables wireless data transmission and real-time display.
